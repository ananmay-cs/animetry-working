# 🎌 Animetry Working

Animetry Working is a personal project I'm building to learn Python, APIs, recommendation systems, and machine learning while creating something around anime.

The idea started as a simple anime recommendation tool and will gradually evolve into a platform that can suggest anime based on genres, ratings, user preferences, and eventually machine learning models.

## Current Features

* Search for an anime title
* Get recommendations based on predefined suggestions
* Simple command-line interface
* Built entirely in Python

## Why I Started This Project

As an anime fan and a computer science student, I wanted a project that would help me practice programming while building something I genuinely enjoy working on.

Instead of following a tutorial, I'm developing this project step by step and learning new concepts as I go.

## Current Status

🚧 Work in Progress

The current version uses a small recommendation database stored in Python dictionaries.
